# 执行概述  
Ames（萨尔莫菌）试验突变原性通常与分子中若干共轭结构或亲电基团密切相关。文献和官方指南一致指出下列结构特征可作为突变原性警报（结构警报），包括：芳香环上的硝基、芳香胺、偶氮基、环氧化物、α,β-不饱和羰基（迈克尔受体）、N-亚硝基、烷基硝酸酯、疏水性杂环、醌类、杂环芳香族和多环芳香烃等。Physicochemical因素如高logP、分子量等虽影响吸收分布，却无公认阈值。下文列出所有重要的Ames突变原性预测规则（简明规则），并在表格中归纳其子结构SMARTS描述、阈值（如有）和文献来源。

1. Check 芳香硝基（Ar-NO₂）【4†L1419-L1422】。  
2. Check 芳香胺（Ar-NH₂/Ar-NHR）【4†L1419-L1422】。  
3. Check 偶氮结构（R–N=N–R）【4†L1419-L1422】。  
4. Check N-亚硝基（R₁R₂N–NO）【4†L1419-L1422】。  
5. Check 环氧化物（三元环氧）【4†L1419-L1422】。  
6. Check 烷基卤化物（R–X）【63†L1673-L1681】。  
7. Check α,β-不饱和羰基（迈克尔受体，C=C–C(=O)）【63†L1691-L1698】。  
8. Check 烷基硝酸酯（R–ONO）【63†L1673-L1681】。  
9. Check 羟肼/肼（R–NH–NH–R）【65†L1775-L1780】。  
10. Check 异氰酸酯（R–N=C=O）【66†L13-L16】。  
11. Check 醌类（邻、对位醌结构）【69†L1743-L1748】。  
12. Check 多环芳香烃（3个及以上融合芳香环）【71†L2009-L2017】。  
13. Check 多环杂芳烃（含异原子的3个及以上融合芳香环）【71†L2019-L2021】。  
14. Calculate logP（无特定阈值）【78†L1-L4】。  

**注：**各规则按文献及机理重要性优先级不等，可分为高（high）、中（medium）、低（low）。如芳香胺、硝基、环氧、迈克尔受体、N-亚硝基、偶氮、醌等属高优先级；烷基卤化物、硝酸酯、羟肼等视具体结构定中优先；多环芳香类属中等偏低优先。
